# Notas de proyecto - Furina Memory Game

## Intencion
Juego de memoria con deteccion de victoria, reinicio, contador y logica testeable.

## Identidad
El proyecto usa una estetica dark/premium con detalles suaves inspirados en interfaces de IA, dashboards y energia gamer/anime moderada. La personalidad aparece en nombres y microcopy, pero el codigo mantiene nombres legibles.

## Flujo principal
1. El usuario abre o ejecuta el proyecto localmente.
2. Completa el flujo principal descrito en el README.
3. Si ocurre un error, el proyecto responde con un mensaje claro y recuperable.
